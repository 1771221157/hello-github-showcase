关于本站

星穹开源

星穹开源 自动同步 HelloGitHub 月刊内容，方便按语言分类浏览和搜索。

数据来源

本站所有项目内容源自 HelloGitHub，由 521xueweihan 创建和维护。

HelloGitHub 每月 28 号发布一期新期刊，每期推荐数十个开源项目，按编程语言分类。

自动更新

通过 GitHub Actions 每天自动检查 HelloGitHub 仓库，有新期刊时自动拉取、解析并重新生成页面。

技术栈

纯静态页面 · Node.js 构建管线 · EJS 模板 · GitHub Actions 部署

功能

  • 按编程语言分类浏览所有项目
  • 完整历史期刊归档，按年份查看
  • 项目搜索过滤
  • GitHub Star 数展示
  • 深色 / 亮色主题切换
  • 响应式布局，适配手机和桌面
